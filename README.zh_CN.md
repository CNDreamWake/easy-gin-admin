## 前置条件

- **[Go](https://go.dev/)** version >= 1.22

## 快速开始

### 更新依赖

```bash
go mod tidy
```

### 创建配置文件

```bash
cp config.default.toml config.toml
```

### 启动

```bash
go run .
```
