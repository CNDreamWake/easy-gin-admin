## Preconditions

- **[Go](https://go.dev/)** version >= 1.22

## Quickly started 

### Update Dependencies

```bash
go mod tidy
```

### Create Configuration File

```bash
cp config.default.toml config.toml
```

### Start the Application

```bash
go run .
```
